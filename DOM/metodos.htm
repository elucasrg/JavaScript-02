Alterando Conteúdo A maneira mais fácil de modificar o conteúdo do HTML é usando a propriedade innerHTML.

document.getElementById(id).innerHTML = new HTML();

Alterando Atributos Podemos alterar também atributos, como por exemplo:

document.getElementById(id).attribute = new value;

Alterando Estilos Para alterar estilos de um elemento HTML usa-se:

document.getElementById(id).style.property = new style; Métodos para remover um elemento do DOM.
Método Descrição
document.createElement(element) Criar um elemento HTML
document.removeChild(element) Remover um elemento HTML
document.appendChild(element) Adicionar um elemento HTML
document.replaceChild(new, old) Substituir um elemento HTML
document.write(text) Escreva no fluxo de saída HTML
